# CYBERKIT

A portable multi-boot cybersecurity and IT recovery toolkit built on a USB drive.

CYBERKIT integrates multiple operating environments and diagnostic tools into a single portable device designed for system recovery, hardware diagnostics, and cybersecurity testing.

---

## Overview

CYBERKIT was created to provide IT professionals and cybersecurity students with a lightweight, portable toolkit capable of:

- Recovering damaged or unbootable systems
- Performing hardware diagnostics
- Conducting basic security testing
- Accessing forensic and recovery utilities

The toolkit is designed to work on most modern x86 systems via USB boot.

---

## Included Environments

| Tool | Purpose |
|-----|------|
| WinPE | Windows recovery and troubleshooting |
| Kali Linux Live | Cybersecurity testing and network analysis |
| Ubuntu Live | General Linux troubleshooting and recovery |
| MemTest86 | Hardware memory diagnostics |
